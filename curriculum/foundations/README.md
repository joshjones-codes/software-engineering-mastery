# Foundations — The Read Track

Saturday-morning reading, paired with whatever spine module you're building. Five book-driven sequences, in order. These are NOT prerequisites for the spine — they run alongside it and slowly change how you think. Each module = chapters + a small build that makes the ideas concrete.

| Stage | Modules | Focus |
|---|---|---|
| [How Computers Work](01-how-computers-work/README.md) | 4 | Book: *Code* (Petzold, 2nd ed). |
| [Programming & Algorithms](02-programming-and-algorithms/README.md) | 7 | Books: *Grokking Algorithms*, then selected *Algorithm Design Manual* chapters. |
| [Operating Systems](03-operating-systems/README.md) | 6 | Book: *Operating Systems: Three Easy Pieces* (free online). |
| [Networking](04-networking/README.md) | 5 | Book: *Computer Networking: A Top-Down Approach* (selected chapters). |
| [Data-Intensive Systems](05-data-intensive-systems/README.md) | 6 | Book: *Designing Data-Intensive Applications*. |
