# Frontend — The Depth Track

You already work here — this track converts working knowledge into top-5-10% depth. Interleave it with the spine: the `must` modules (browser internals, React internals, performance) are senior-frontend interview material *now*; the offline/device-API modules pair directly with SlipCheck field-crew features. Concepts over frameworks: the browser is the platform, React is one library on it.
